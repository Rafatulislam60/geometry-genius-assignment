1. What is DOM and what is the purpose of DOM?

a. Document Object Model is referred to by the shorthand DOM. It is the internal representation of a web page by a web browser. What material belongs on a page and how each element of that content interacts with other elements are decided by the DOM.

b. Maintaining a standardized, language-independent programming interface for HTML and XML documents is one of the key goals of the DOM's development. By establishing and upholding this standard, programmers can use any language or development environment to generate, navigate, and alter these kinds of documents' contents predictably.

2. How will you select HTML elements using DOM? Name the DOM methods.

Through the 5 ways I can select HTML elements using DOM. These are,

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

3. What is event bubble?

One form of event propagation is Event bubbling in JavaScript. Prior to triggering the outermost element in the hierarchy, the event first activates the innermost target element and then successively the target element's parent element.

What is a callback function and why will you use it?

a. Any function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function.

b. Callbacks are used to manage the outcomes of asynchronous actions, which don't disrupt the rest of the program's execution. Instead, the program keeps running, and when the operation is finished, the callback function is called.
